## 📝 TIPS UNTUK QUERY YANG EFEKTIF

### ✅ DO (Lakukan)
- Spesifik: "Prediksi DBD di Kei Kecil Maret 2026" vs "Prediksi DBD"
- Kontekstual: Sebutkan kecamatan, periode, program spesifik
- Terstruktur: Gunakan format pertanyaan yang jelas
- Follow-up: Tanya detail lebih lanjut untuk insights lebih dalam

### ❌ DON'T (Hindari)
- Terlalu umum: "Bagaimana kesehatan di Maluku Tenggara?"
- Ambigu: "Berapa angkanya?" (angka apa?)
- Multiple questions: Pisahkan jadi beberapa query
- Tanpa konteks: Sebutkan periode waktu dan lokasi

### 🎯 Best Practices
1. Mulai dengan query sederhana, lalu detail
2. Gunakan follow-up questions untuk deep dive
3. Minta visualisasi atau format output spesifik
4. Verifikasi hasil dengan data source
